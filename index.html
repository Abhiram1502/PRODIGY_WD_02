How to use

  • Start the Stopwatch:

    • Click the Start button to begin the timer. The stopwatch will start counting from 00:00:00:00.
    The seconds and milliseconds will update every 10ms.

  • Stop the Stopwatch:

    • Click the Stop button to pause the stopwatch. The time will freeze, and the stopwatch will stop counting.
    If you wish to resume, click the Start button again.

  • Reset the Stopwatch:

    • Click the Reset button to clear the stopwatch and set the time back to 00:00:00:00.
    This will reset the timer to its initial state.
